# PUBG

## Player Unknown Battlegrounds command

## Usage

`<prefix>pubg <username> <type> [lastXgames]`

* For the \[lastXgames\], you can only go up to the last five games 

**Types:**

* Solo
* Duo
* Trio
* Squad

## Example

g!pubg Moondye7 solo last1
