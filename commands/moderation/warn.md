# Warn Command

### Usage

`<prefix>warn [action] [user] [reason] [whatever]`

### Alias

* Use the alias _**w**_ for _**warn**_

### Example

`g!warn add @caligula Test`  
`g!warn list @caligula`  
`g!warn remove @caligula`  
`g!warn removeall @caligula`
